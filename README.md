🎓 PM Internship Recommendation System

A smart recommendation system that suggests suitable internships to students based on their skills using CountVectorizer and Cosine Similarity. The project includes an interactive web interface built with Streamlit.

🚀 Features

🎯 Personalized internship recommendations
🧠 Content-based filtering approach
⚡ Fast similarity matching using cosine similarity
🌐 Interactive UI using Streamlit
🔍 Skill-based matching system

🛠️ Tech Stack

Programming Language: Python
Machine Learning: scikit-learn
Vectorization: CountVectorizer
Similarity Measure: Cosine Similarity
Frontend/UI: Streamlit
